Project Parental Discounts

Overview:

The overview of this project is to create a GUI based program that allows new or expectant parents to enter their location, select the items they‚Äôd like to be alerted for discounts, and have a custom made alert sent to their devices whenever these items are marked on special. This will therefore make online shopping for new parents less stressful, as their chosen items (for example, baby formula) and brands will be scouring online resources like Amazon or Coles for marked discounts, alerting them when they‚Äôre available for discounted purchase. This will overall contain basic parental needs such as prams, baby formula, baby clothes (gender neutral or specific) ,toys, bassinets, car seats, wipes and nappies.

Motivation:

As a father of one (soon to be two) I often find myself having to purchase baby related items on a whim, such as nappies, wipes or formula. While seeing discounts in store is useful, the ability to know in advance when these items are on special would be much more handy, especially if local stores are stocking only at full price. This program would seek to help resolve this stress, and ease the plight of single or struggling parents, to whom financial woes are extremely dangerous. In a world of decreasing stability and ever increasing crises, the ability to know when essential baby products are affordable will be a huge relief to those currently undergoing financial stress.

Specifications:

First, the user would download the program, and be shown a Guided User Interface (GUI) that asks them to enter their location (country, state, city/postcode). The user is then presented with another checkbox displaying ‚ÄúOnline‚Äù, which when indicated, includes online offers as well. Afterwards, the user is taken to the second screen, which has a list of items they may select for notification. These items include but are not limited to:

  • Strollers/prams
  • Nappies/diapers
  • Wipes
  • Formula
  • Clothing
  • Bottles
  • Toys

After the user has selected the items they wish to be notified on (some options, when selected, may prompt the user to add more specifications, such as gender specific clothing or brand specific formula), they will then be prompted as to how often they would like to receive notifications on these specials. These options will be anything from daily, weekly, fortnightly or monthly. Consideration will be taken in the future for adding different notification options for specific items such as formula, or toys, as needs shift.

Once these options have been selected, the user will then be taken to the contact details screen, which will capture their contact information in order to send them the notifications of when these items go on special.

The user should also be sent a verification email to ensure that they have received the correct notification.

In future, there may be additional features added such as the ability to later customise the notifications (remove items), change locations, change specific stores, request for stores to be added, et cetera.

Skills Required:

In order to undertake the creation of this project, an intermediate to advanced level of programming will be required, not only to create the frontend of the application but also the backend. Security measures will need to be put in place to protect the information of the users, so an understanding of these protocols is also required.

An understanding of languages such as Python or C# would be most beneficial for these tasks, as these languages offer solid automation and project scaling, respectively.

Database quering, the likes of MYSQL would be a mandatory requirement in order to store the information of the users, their preferences, contact information, location, et cetera.

Networking and security knowledge are vital for the success of this project, as enough scaling would need to be done to ensure there is always room for more users, and that all users private information would be safely secured in a non-accessible way, which will require a sound understanding of web applications, databases, and server security measures and protocols. In future, if deployed to a mobile application, these measures and protocols will need to extend to mobile devices as well.

Outcomes:

The overall outcome and goal of this project is that the users of the program are able to rest more easily knowing that there are many affordable options in the world that they no longer have to search for. In the modern world, many people are time poor due to financial constraints, having the ability to simply open a program or their own emails and see the latest discounts on essential items would truly be much less burdensome to new or struggling parents. To be able to select, within a few minutes, which items are needed now or in the future, at affordable rates, would hopefully create less stress at home and within themselves, allowing them to enjoy more thoroughly the benefits of parenthood.
